# Deep Haven — 隐私政策

## 数据收集

Deep Haven 是一款本地运行的末日生存游戏。

- **Phase 1 (CLI)**: 不收集任何数据，不联网。
- **Phase 2 (Web SPA)**: 游戏数据存储在浏览器 localStorage 中，不上传服务器。
- **Phase 3 (全栈)**: 如果需要联网功能（AI 文本生成等），将在此更新隐私政策。

## 第三方服务

- 后续如接入 LLM API（如 DeepSeek），用户可自行配置 API Key。
- 不会在服务端存储用户的 API Key。

## 联系方式

如有隐私相关问题，请通过 GitHub Issues 联系：https://github.com/liumian1201/DeepHaven/issues
